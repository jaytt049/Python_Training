## Topic Plan

### 1) Variables and Data Types
Assignments:
- Store and print user details (name, age, salary)
- Type checker (identify type of input)

### 2) Mathematical Operations
Assignments:
- Calculator
- Simple interest calculator
- Unit converter (km to miles, Celsius to Fahrenheit)

### 3) Conditional Statements
Assignments:
- Even/odd checker
- Grade calculator
- Largest of three numbers

### 4) Loops
Assignments:
- Multiplication table generator
- Factorial calculator
- Prime number checker

### 5) Strings
Assignments:
- Palindrome checker
- Vowel counter
- Word frequency counter

### 6) Lists
Assignments:
- Remove duplicates from list
- Find second largest number
- Merge two lists without duplicates

### 7) Dictionaries
Assignments:
- Student marks manager
- Word count dictionary
- Frequency counter (characters in string)

### 8) Functions
Assignments:
- Modular calculator using functions
- Recursive factorial
- Fibonacci using function

### 9) File Handling
Assignments:
- Read/write text file
- Log analyzer (error count)
- Store and retrieve contacts from file

### 10) Exception Handling
Assignments:
- Safe calculator with error handling
- File reader with exception handling
- Input validator (only numbers allowed)

### 11) List Comprehension and Lambda
Assignments:
- Square numbers using list comprehension
- Filter even numbers using lambda
- Map operation on list (double values)

### 12) APIs (requests library)
Assignments:
- Fetch data from public API
- Weather fetch script
- Retry mechanism on failed API call

### 13) Object-Oriented Programming (OOP)
Assignments:
- Student class with grades
- Product and inventory system

### 14) Modules and Packages
Assignments:
- Create custom utility module
- Split calculator into multiple files
- Import and reuse functions
